# TinyML 学習計画 (O'Reilly TinyML, ISBN: 978-1-492-05204-3)

## 目的

- マイコン上で動く機械学習の全体像を理解する
- 本に出てくる主要なプロジェクト（ウェイクワード検出、画像、モーション、異常検知など）を自分の手で再現する
- 自分のボード/センサに応用できる状態にする

## 使用する本

- O'Reilly 『TinyML: Machine Learning with TensorFlow Lite on Arduino and Ultra-Low-Power Microcontrollers』
  - ISBN: 978-1-492-05204-3

## 進め方（例）

- 平日: 1 日 30〜60 分、1 章の一部分ずつ進める
- 週末: 実機へのデプロイや実験コードをまとめて行う
